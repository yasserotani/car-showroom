# Car Showroom OpenGL Project

## Requirements
- Windows
- Visual Studio 2022
- CMake 3.20+
- OpenGL-compatible GPU

## Build & Run

```powershell
git clone https://github.com/USERNAME/CarShowroom2025.git
cd CarShowroom2025

cmake -S . -B build -G "Visual Studio 17 2022"
cmake --build build --config Release

.\build\Release\CarShowroom2025.exe
